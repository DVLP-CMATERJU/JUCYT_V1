# JUCYT_V1 Cytology Dataset

The dataset contains 10 cytology image dataset
  * 5 Benign samples
  * 5 Malignant samples


The cytology images are captured by trinacular microscope along with 10 MP CMOS Camera. 
The microscopic cytology images are 960X1280 pixels in dimension. These 10 images are captured in 40x magnification.
